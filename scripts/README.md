# Smart Tred - فایل‌های راه‌اندازی

راه‌اندازی و مدیریت اپلیکیشن Smart Tred

## 🚀 شروع سریع

**Windows:**
```bash
scripts\run.bat
```

**Mac/Linux:**
```bash
bash scripts/run.sh
```

**Docker:**
```bash
docker-compose up
```

## 🔧 تنظیمات خودکار

برای تنظیم خودکار Backend و Frontend:

**Windows:**
```bash
python scripts/setup.py
```

**Mac/Linux:**
```bash
python3 scripts/setup.py
```

## 📁 ساختار فایل‌ها

```
scripts/
├── run.bat          # راه‌انداز Windows
├── run.sh           # راه‌انداز Mac/Linux
└── setup.py         # تنظیم خودکار
```
