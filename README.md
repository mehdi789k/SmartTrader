# 🚀 Smart Tred - MetaTrader5 Web Interface

> Professional web application for connecting to MetaTrader5 and managing trading accounts

## ⚡ Quick Start

Choose your platform:

### Windows
```bash
scripts\run.bat
```

### Mac/Linux  
```bash
bash scripts/run.sh
```

> The launchers now create a Python virtual environment and install the backend package dependencies automatically.

### Docker
```bash
docker-compose up
```

Then open your browser: **http://localhost:3000**

## 🔐 Security

⚠️ For production:
- Change `SECRET_KEY` in environment variables
- Use HTTPS only
- Update `CORS_ORIGINS` for your domain  
- Implement rate limiting
- Keep dependencies updated
